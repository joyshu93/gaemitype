# Result Image Prompts

개미타입 결과 이미지 16종을 생성할 때 사용한 원본 프롬프트 기록 문서.

용도:

- 추후 이미지 재생성
- 스타일 일관성 유지
- 외주/재작업 시 레퍼런스 전달
- OG 이미지 또는 결과 카드 확장 작업 시 기준 문서

주의:

- 이 문서의 타입 한글명은 이미지 생성 당시 네이밍 기준이다.
- 현재 서비스 코드 안의 타입명과 일부 다를 수 있으므로, 실제 연결 기준은 항상 `4글자 타입 코드`를 우선한다.
- 파일 매핑 기준은 `ALRD.png`, `ALRC.png`처럼 타입 코드 기반으로 유지한다.
